============================================================
  CÓMO LEVANTAR LA PÁGINA WEB (LANDING PAGE)
  Proyecto: HelpingApp - Landing Page
============================================================

Requisitos previos:
  - Windows 10/11 (o cualquier SO con Node.js)
  - Node.js (versión 18 o superior)
  - NPM (se instala automáticamente con Node.js)

------------------------------------------------------------
PASO 1: Instalar Node.js
------------------------------------------------------------
1. Ve a https://nodejs.org/
2. Descarga la versión LTS (recomendada)
3. Ejecuta el instalador y sigue los pasos (siguiente, siguiente...)
4. Una vez instalado, abre una terminal (CMD o PowerShell) y
   verifica que funciona:
       node --version
       npm --version
   Deberías ver números de versión (ej: v22.x.x y 11.x.x)

------------------------------------------------------------
PASO 2: Colocar el proyecto en el ordenador
------------------------------------------------------------
- Copia la carpeta "landing-page" completa a la ubicación
  que quieras en tu ordenador.
- Ejemplo: C:\Users\TuUsuario\landing-page

------------------------------------------------------------
PASO 3: Abrir terminal en la carpeta del proyecto
------------------------------------------------------------
- Abre PowerShell o CMD
- Navega hasta la carpeta:
    cd ruta\donde\copiaste\landing-page
  Ejemplo:
    cd C:\Users\TuUsuario\landing-page

------------------------------------------------------------
PASO 4: Instalar dependencias
------------------------------------------------------------
Ejecuta:
    npm install
Esto descargará todas las librerías necesarias (React, Vite,
etc.). Tarda entre 30 segundos y 2 minutos.

------------------------------------------------------------
PASO 5: Iniciar el servidor de desarrollo
------------------------------------------------------------
Ejecuta:
    npm run dev
Verás un mensaje como:
    VITE v8.x.x  ready in Xms
    Local: http://localhost:5173/

Abre ese enlace en tu navegador.

------------------------------------------------------------
PASO 6: Detener el servidor
------------------------------------------------------------
En la terminal, pulsa Ctrl + C para detenerlo.

------------------------------------------------------------
COMANDOS ÚTILES
------------------------------------------------------------
  npm run dev      -> Inicia el servidor de desarrollo
  npm run build    -> Genera la versión de producción en /dist
  npm run preview  -> Previsualiza la versión compilada
  npm run lint     -> Revisa errores de código

------------------------------------------------------------
ESTRUCTURA DE CARPETAS (resumen)
------------------------------------------------------------
  landing-page/
    index.html          -> Punto de entrada
    src/                -> Código fuente (React)
    public/             -> Archivos estáticos
    dist/               -> Versión compilada (para producción)
    package.json        -> Configuración y scripts
    vite.config.js      -> Configuración de Vite
    node_modules/       -> Dependencias (no tocar)

------------------------------------------------------------
NOTAS
------------------------------------------------------------
- La carpeta node_modules/ NO se debe copiar/pegar. Siempre
  se regenera con "npm install".
- Para distribución en producción, ejecuta "npm run build"
  y sube la carpeta "dist/" a un servidor web.
- Si ves errores de permisos en Windows, abre la terminal
  como Administrador.
